# CSV-Chataigne-module
Module to import CSV files in Chataigne : https://benjamin.kuperberg.fr/chataigne
Let production do the work and let them take responsibility for the content they send.

The module will parse the csv file and allow you to broadcast the datas inside one row after the other.

![Example](https://github.com/etehel/CSV-Chataigne-Module/assets/40066203/cc1df683-8da1-4f7e-bc34-934e2319a84d)
# Installation
Download it then place the folder inside your `<Documents>/Chataigne/modules` folder.
